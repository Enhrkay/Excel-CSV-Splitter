# Excel/CSV Splitter by Column

This is a Python Tkinter application that allows you to:
- Load Excel or CSV files
- Preview unique values in a column
- Split the file automatically by a selected column
- Normalize Nigerian phone numbers

## Requirements
- Python 3.x
- pandas
- tkinter (usually included with Python)
- re, os (standard libraries)

## How to Use
1. Run `CSV Splitter.pyw` (double-click to avoid CMD window popping up)
2. Select your Excel/CSV file
3. Choose the sheet (for Excel)
4. Select the column to split by
5. Preview unique values
6. Click "Split File" to generate separate files
